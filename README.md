# cogno-cli-releases

Public release artifacts for the [cogno CLI](https://cogno.studio).

This repo contains no source code — it exists to make GitHub Releases publicly
downloadable. Source lives in a separate private repo.

## Install

```sh
curl -fsSL https://cogno.studio/install.sh | sh
```

## Binaries

See the [Releases page](https://github.com/cogni-engine/cogno-cli-releases/releases)
for prebuilt binaries (Darwin, Linux, Windows × amd64, arm64).
